# Mental Health Analysis Using Social Media

## Overview
This project analyzes the impact of social media on mental health using Twitter data, survey responses, sentiment analysis, and Power BI dashboards.

## Technologies Used
- Python
- Power BI
- Pandas
- NLP
- Sentiment Analysis

## Features
- Tweet analysis
- Sentiment analysis
- Correlation analysis
- Power BI visualization
- Mental health insights

## Dataset
- Twitter data
- SMMH Kaggle dataset
- Survey responses

## Dashboard
Power BI dashboard used to visualize:
- Sleep patterns
- Self-esteem
- Anxiety levels
- Concentration issues
- Social media usage trends

## Conclusion
The project shows that excessive social media usage negatively impacts mental health, especially sleep quality, concentration, and self-esteem.
